# Data-Projects
Data analytics projects
Sales data analysis project:
  My first step was I uploaded the dataset to excel and created a pivot table showing the sales per region and the sales per category.
  Next, I analyzed the sales dataset using SQL to evaluate regional and category performance.
  I also used different functions such as SUM, AVG and COUNT to identify trends and patterns in the data. 
  Using this information I uploaded the data into Power BI and created an interactive dashboard to visualize sales performance.
  With these tools I was able to identify key insights including top performing regions and category trends as well as identifying how the trends differ on a regional basis.
